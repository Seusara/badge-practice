# badge-practice

Practice repo for learning Git/GitHub PR workflows.

## Project structure

- `index.js` - entry point, prints a greeting and a couple of math examples
- `math.js` - small math utilities (`add`, `subtract`, `multiply`, `divide`)
- `test.js` - basic assertion tests for the functions above

## Usage

```bash
npm start          # runs index.js, greets "World"
node index.js Ana  # greets a custom name instead
npm test           # runs test.js
```

## Example

```bash
$ npm start
Hello, World!
2 + 3 = 5
2 * 3 = 6
```
